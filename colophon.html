Colophon

A colophon is a brief statement about the production of a book - who made it, how it was made, and what tools were used.

About This Site

The aesthetic is inspired by neon bar signs - the kind you'd find in a dive bar that feels like home. Warm, a little rough around the edges, but authentic. The colors glow, the vibe is casual, and there's no pretense.

Technology Stack

Built With

  • HTML5 - Semantic markup for structure
  • CSS3 - Custom design with CSS variables, grid, flexbox, and animations
  • Vanilla JavaScript - Subtle animations (parallax, neon flicker, fade-ins) with accessibility support

Typography

  • Caveat by Pablo Impallari - Handwritten font for neon sign headings
  • Asar - Readable body font with character

Services & Tools

  • Formspree - Contact form handling
  • Simple Icons - Icon font for social links
  • Claude Code - AI pair programming assistant for development

Design Philosophy

Cool Slate Aesthetic

The site uses a cool charcoal/slate color palette as a base with pops of neon color. This creates a modern, electric vibe while keeping the neon dive bar aesthetic. The four neon colors (pink, amber, green, cyan) rotate throughout the design to add visual interest.

Animation Details

Animations are subtle and purposeful - neon signs flicker like real neon, elements fade in as you scroll, and the background has a gentle parallax effect. There are also a few hidden easter eggs for the curious (try the Konami code, or triple-click the logo).
